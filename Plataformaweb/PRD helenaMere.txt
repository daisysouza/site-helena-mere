Design a premium responsive real estate platform for a Brazilian realtor named Helena Mêre.

Project Type:
Boutique real estate portal focused on property showcase and lead generation.

Brand Personality:
Elegant, trustworthy, warm, human-centered, sophisticated but accessible.

Important:
This is NOT a corporate real estate marketplace.
The brand should feel personal, boutique, welcoming, and emotionally connected to the idea of home.

Visual References:
Use a mix of:

1. Modern real estate portals like Lara Garcia Corretora
2. Organic emotional branding inspired by nature

Brand Elements:

* Green tones
* Soft neutral backgrounds
* Spacious layouts
* Premium photography
* Nature-inspired accents (subtle flowers, organic curves, warm lighting)

Color Palette:
Primary Green: #028940
Secondary Green: #6EA24C
Light Accent Green: #B1D182
Background: #F4F1E9
White: #FFFFFF

Typography:
Headings: Playfair Display (luxury serif)
Body: Inter (modern sans serif)

Target Audience:
Brazilian families and investors aged 35–65 seeking homes, apartments, and premium properties in Contagem, Cabral, and Belo Horizonte.

Main Goals:

* Build trust
* Showcase properties
* Generate WhatsApp leads
* Improve local SEO
* Strengthen personal brand authority

Pages Required:

1. HOME
   Sections:

* Hero banner with luxury property photo

* Headline:
  “O lugar certo para viver começa com a escolha certa.”

* CTA buttons:
  View Properties
  Contact on WhatsApp

* Search bar section:
  Property type
  Neighborhood
  Price range
  Bedrooms

* Featured properties grid

* About Helena section with professional portrait

* Benefits section

* Testimonials carousel

* Final CTA

2. PROPERTIES LISTING
   Features:

* Advanced filters
* Grid layout
* Modern property cards
* Sorting options
* Pagination

Property card includes:

* Photo
* Price
* Neighborhood
* Bedrooms
* Bathrooms
* Parking
* Square meters
* CTA button

3. PROPERTY DETAIL PAGE
   Sections:

* Large gallery
* Property specs
* Description
* Amenities
* Location map
* Sticky WhatsApp CTA
* Similar properties

4. ABOUT PAGE

* Professional portrait
* Story
* Mission
* Values
* Trust signals

5. LIST YOUR PROPERTY PAGE
   Lead generation form:

* Name
* Phone
* Property type
* Neighborhood
* Estimated value

6. CONTACT PAGE

* WhatsApp
* Instagram
* Contact form
* Map
* Social links

UI Requirements:

* Clean luxury aesthetic
* Mobile-first
* Large whitespace
* Smooth rounded cards
* Soft shadows
* Premium subtle animations
* Sticky header
* Floating WhatsApp button

Avoid:

* Overly corporate style
* Heavy dark UI
* Generic marketplace visuals
* Cluttered layouts
